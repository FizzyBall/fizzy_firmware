The code in `sensor_fusion_9x` is based on the ispu-examples from STM
https://github.com/STMicroelectronics/ispu-examples

NOTE: To compile ispu source code, a the designated compiler toolchain from ST () needs to be setup.

After compilation, the resulting `ispu.h` file needs to be copied to `components/mpu/sensor_fusion_9x.h`

Some notes for fine tuning:
https://github.com/STMicroelectronics/ispu-examples/issues/1#issuecomment-2460036704
